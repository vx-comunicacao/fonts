guia — entenda as fontes variáveis

O que são fontes variáveis?

Uma fonte variável (Variable Font, ou VF) é um único arquivo que guarda toda uma família tipográfica de forma contínua. Em vez de um arquivo fechado para cada peso (Light, Regular, Bold…), ela armazena os extremos do desenho e deixa o navegador interpolar qualquer valor entre eles.

Fonte estática (tradicional)

  • 1 arquivo por peso/estilo (ex.: 14 arquivos)
  • Só os pesos "prontos": 400, 700…
  • Mais requisições de rede
  • Nada entre um peso e outro

Fonte variável

  • 1 arquivo cobre a família inteira
  • Qualquer peso do intervalo (ex.: 437, 650…)
  • Menos requisições
  • Transições/animações suaves de peso

Eixos (axes)

O que "varia" na fonte são os eixos. Cada eixo tem uma tag de 4 letras e um intervalo mínimo–máximo. Existem eixos registrados (padrão) e eixos personalizados (definidos pela fundição). As fontes Ipiranga VF expõem wght e ital:

wght Weight / Peso — controla a espessura. Ipiranga: 200 a 900. Mapeado pela propriedade CSS font-weight.
ital Italic / Itálico — alterna entre romano (0) e itálico (1). Ipiranga: 0 a 1. Ativado por font-style: italic.
slnt Slant / Inclinação — inclina as letras em graus (oblíquo). Não presente na Ipiranga; mapeado por font-style: oblique <ângulo>.
wdth Width / Largura — condensada ↔ expandida. Não presente na Ipiranga; mapeado por font-stretch.
opsz Optical Size — ajusta o desenho ao tamanho de uso. Não presente na Ipiranga; mapeado por font-optical-sizing.

CSS para fontes variáveis

1. Declarar o @font-face com intervalo

A diferença para a fonte estática é que font-weight recebe dois valores (o intervalo do eixo), não um só. Para o itálico, declara-se um segundo @font-face apontando para o mesmo arquivo com font-style: italic: 

@font-face {
    font-family: 'Ipiranga Titulos VF';
    src: url('variable/IpirangaVariableTitulos.woff2') format('woff2');
    font-weight: 200 900;        /* intervalo do eixo wght */
    font-style: normal;
    font-display: swap;
}
@font-face {
    font-family: 'Ipiranga Titulos VF';
    src: url('variable/IpirangaVariableTitulos.woff2') format('woff2');
    font-weight: 200 900;
    font-style: italic;         /* ativa o eixo ital */
    font-display: swap;
}

2. Usar com propriedades CSS normais

O jeito recomendado é usar as propriedades de alto nível — elas mapeiam para os eixos automaticamente e continuam funcionando se um dia a fonte voltar a ser estática: 

.titulo {
    font-family: 'Ipiranga Titulos VF', sans-serif;
    font-weight: 650;      /* qualquer valor entre 200 e 900 */
}
.titulo em { font-style: italic; }

3. Controle fino com font-variation-settings

Para acessar eixos diretamente (ou eixos personalizados sem propriedade equivalente), use font-variation-settings com a tag do eixo. Prefira as propriedades do passo 2 quando existirem; recorra a isto só quando necessário: 

.titulo {
    font-variation-settings: 'wght' 650, 'ital' 0;
}
Atenção: font-variation-settings não é herdado de forma acumulativa — ao redefini-lo, repita todos os eixos que quiser manter, senão eles voltam ao valor padrão.

4. Animar / transicionar o peso

Como o eixo é contínuo, dá para animar o peso de forma fluida:

.titulo {
    font-family: 'Ipiranga Titulos VF', sans-serif;
    font-weight: 400;
    transition: font-weight .3s ease;
}
.titulo:hover { font-weight: 800; }
Suporte: fontes variáveis funcionam em todos os navegadores modernos (Chrome, Edge, Firefox, Safari). A lista de font-family já inclui a versão estática como fallback ('Ipiranga Titulos VF', 'Ipiranga Titulos').
integração — como usar

Usar em outros projetos

Publique a pasta iprg/ inteira (com text/, title/, variable/ e os arquivos .css) na sua CDN, mantendo a estrutura de subpastas. Depois é só importar.

Escolher o arquivo — qual e quando usar

A pasta traz vários .css hardcoded (sem @import). Carregue só o que o projeto usa: menos CSS para parsear e sem baixar declarações de fontes que nunca serão aplicadas.

ArquivoContémQuando usar
ipiranga-titulos.css Só Títulos (estática) O projeto usa apenas títulos/display.
ipiranga-textos.css Só Textos (estática) O projeto usa apenas corpo de texto.
ipiranga-estaticas.css Títulos + Textos (estáticas) Caso mais comum: as duas famílias, sem variável.
ipiranga-variable.css Só as variáveis (VF) Precisa de pesos intermediários/animação, ou quer 1 arquivo por família.
stylesheet.css Tudo (estáticas + VF) Protótipo ou quando quer todas as opções disponíveis.

Importar

Via <link> no <head> (troque SUA-CDN). Ex.: as duas famílias estáticas:

<link rel="stylesheet" href="https://SUA-CDN/iprg/ipiranga-estaticas.css">

Ou via @import no topo do seu CSS:

@import url('https://SUA-CDN/iprg/ipiranga-estaticas.css');

Aplicar as fontes

Cada família usa um único font-family; o peso e o itálico são controlados por font-weight / font-style. Pesos: 100 Thin · 300 Light · 400 Regular · 500 Medium · 700 Bold · 800 ExtraBold · 900 Black. 

/* Títulos — display / chamadas */
h1, h2, h3 {
    font-family: 'Ipiranga Titulos', sans-serif;
    font-weight: 900;
}

/* Textos — corpo / interface */
body, p {
    font-family: 'Ipiranga Textos', sans-serif;
    font-weight: 400;
}

/* Itálico */
em { font-style: italic; }

Fontes variáveis (opcional)

Eixo de peso contínuo de 200 a 900 + itálico, em um único arquivo por família. Substituem as estáticas por completo. 

.titulo-fluido {
    font-family: 'Ipiranga Titulos VF', 'Ipiranga Titulos', sans-serif;
    font-weight: 650;                /* qualquer valor 200–900 */
}
.titulo-fluido em { font-style: italic; }   /* ativa o eixo ital */

Performance (opcional)

Pré-carregue os arquivos mais usados no <head> para evitar flash:

<link rel="preload" as="font" type="font/woff2" crossorigin
      href="https://SUA-CDN/iprg/text/IpirangaTextos-Regular.woff2">
<link rel="preload" as="font" type="font/woff2" crossorigin
      href="https://SUA-CDN/iprg/title/IpirangaTitulos-Black.woff2">
